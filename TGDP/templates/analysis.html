OVERALL - ANALYSIS








The main purpose of this project is to analyze how different factors affect the GSDP and GSDP per capita of our country India and each other.

Some things we will find out in this analysis are

  • Where does each state stand when ranked in these factors?
  • Can India still be called an agricultural country?
  • Which sector is contributing the most to our GDP?
  • How true is the saying "Money can't buy happiness"?
